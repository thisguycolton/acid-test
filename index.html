Co-Host Best Practices

NOTE: The host or cohost has the authority and responsibility, during the meeting, to act as necessary to ensure a safe space for those present. Examples: To mute and/or communicate with person via chat window and/or remove a person/member or put a person/member in the waiting room to address offensive, harassing and/or harmful behavior.

BEFORE THE MEETING

Log in 15 minutes prior to the meeting.
Claim Yourself as the “Host”
- If you arrive first, CLAIM HOST (please refer to the meeting host or Host Coordinator for the Host key #)
Click “PARTICIPANTS” button on the bottom panel
Click “CLAIM HOST:
Give Chair Person (Host, Secretary) “Co-Host” capabilities
Click the 3 dots on the participant “square”
Select “MAKE CO-HOST”

USER VIDEO

- If it is necessary to ask a participant to start their video, send them a request through the window by clicking on the three dots and select “Ask to Start Video”:
- If you receive the following notification: "Zoom User will start video later"
- Message the participant privately
- If participant refuses, or if they become combative, use discretion. Closely monitor the individual or remove them from the room.

MANAGING ZOOM BOMBERS

In the event of a Zoom Bombing, please following the below Acid Test approved protocol:
1. MUTE all - Select the options disabling participants ability to unmute themselves.
2. Remove bomber(s) from the meeting by clicking MORE next to their name in the participants section
3. Either place them in the waiting room - OR - remove them
